# AI SEO Blog & Content Cluster Generator

## Project Overview

The AI SEO Blog & Content Cluster Generator is an AI-powered content creation project designed to generate SEO-friendly blog content and content clusters for local businesses.

This project uses Prompt Engineering and Artificial Intelligence to create:

- SEO Keyword Strategy
- Meta Titles & Descriptions
- Pillar Blog Outlines
- Long-form Pillar Blog Content
- Supporting Blog Topics
- Internal Linking Suggestions
- Local SEO Recommendations

The objective is to help businesses improve their online visibility, search engine rankings, and content marketing efforts.

## Prompt Logic

The prompt accepts the following inputs:

- Business Name
- Business Type
- Location

Based on these inputs, AI generates:

1. Primary Keyword
2. Secondary Keywords
3. Search Intent
4. SEO Meta Title
5. SEO Meta Description
6. Pillar Blog Outline
7. 1200+ Word SEO Blog
8. Supporting Blog Topics
9. Internal Linking Suggestions
10. Local SEO Recommendations
